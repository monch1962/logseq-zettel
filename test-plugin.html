    Usage Instructions

    Slash Commands:

    • /zettel fleeting - Create a fleeting note
    • /zettel literature - Create a literature note
    • /zettel permanent - Create a permanent note
    • /zettel - Show available commands

    Mobile Shortcuts:

    • /z fle - Fleeting note
    • /z lit - Literature note
    • /z per - Permanent note

    Expected Workflow:

    1. Use slash command to create note
    2. Note is created with timestamp ID as page name
    3. Note appears in daily journal's Zettelkasten section
    4. Use Syncthing to sync between devices
    5. Manually copy note files to Obsidian vault
